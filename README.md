# Movie Search App

A simple web application that allows users to search for movies and view key details such as title, rating, genre, plot, and poster.

## Website
https://movie-search-app-jfeytbszdqbgrzvxuwpkqk.streamlit.app/


## Features

* Search for any movie by name
* Displays:

  * Title
  * Year
  * IMDb Rating
  * Plot
  * Poster Image

## Tech Stack

* Python
* Streamlit
* OMDb API

## Run Locally

### 1. Clone the repository

```bash
git clone https://github.com/dineshrishim/movie-search-app.git
cd movie_app
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the app

```bash
streamlit run movie_app.py
```
